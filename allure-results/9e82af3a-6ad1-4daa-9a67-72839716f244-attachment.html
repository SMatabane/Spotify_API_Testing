POST to https://api.spotify.com/v1/playlists/invalid/tracks

Body

{
    "uris": [
        "spotify:track:INVALID_TRACK_1234",
        "spotify:track:INVALID_TRACK_5678"
    ],
    "position": 0
}

Headers

Authorization: valid
Accept: */*
Content-Type: application/json

Curl

curl -v -X POST 'https://api.spotify.com/v1/playlists/invalid/tracks' -H 'Authorization: valid' -H 'Accept: */*' -H 'Content-Type: application/json' -d '{ "uris": [ "spotify:track:INVALID_TRACK_1234", "spotify:track:INVALID_TRACK_5678" ], "position": 0 }'
